## **总体功能概述**

这个程序：

1. 从文件 `graph.in` 读取两个整数 `n` 和 `m`。
    
2. 通过两个复杂的 DP（动态规划）计算两个数组 `f` 和 `f1`，这两个 DP 结构有点像“背包问题”或“分拆计数”，并且涉及平方根分界。
    
3. 计算
    
    ans=msum over i f[i]×f1[n−i]  (md−1)(modmd)ans = m^{\text{sum over i} \, f[i] \times f1[n-i] \bmod (md-1)} \pmod{md}
    
    （这里 `md = 999999599`，是个质数附近的数；指数对 `md-1` 取模，是因为费马小定理。）
    
4. 输出 `ans` 到 `graph.out`。
    

它的核心其实是一个分块（sqrt 分界）的组合计数问题，用两个 DP 分别处理“小于等于 sqrt(n)”和“大于 sqrt(n)”的部分，然后通过指数运算得到最终结果。

### 第一个 DP（计算 f）

```cpp
auto f=[](auto f,auto b,auto _)->std::vector<int>{
    f[0]=1;
    return [](auto it,auto f,auto _)->std::vector<int>{
        std::for_each(it.begin(),it.end(),[&](auto i){
            [&](auto it){
                std::for_each(it.begin(),it.end(),[&](auto j){
                    [&](auto &x,auto y){
                        return x=(x+y>=md-1?x+y-md+1:x+y),void();
                    }(f[j],f[j-i]);
                });
            }([](auto _,auto __)->std::vector<int>{
                return std::iota(_.begin(),_.end(),__),_;
            }(std::vector<int>(_-i+1),i));
        });
        return f;
    }([](auto _,auto __)->std::vector<int>{
        return std::iota(_.begin(),_.end(),__),_;
    }(std::vector<int>(b),1),f,_);
}(
    std::vector<int>(mxn,0),
    (int)sqrt(*_),*_
);
```

这个部分干的事：

1. `f` 是大小为 `mxn` 的 DP 数组，初始化为 0，`f[0]=1`。
    
2. 外层参数 `b = floor(sqrt(n))`，`_ = n`。
    
3. 构造一个 `it = [1, 2, ..., b-1]` 的列表，依次做**完全背包**：
    
    - 对每个物品 `i`（1 到 b-1），遍历容量 `j`（从 `i` 到 n），更新：
        
        f[j]=f[j]+f[j−i](mod md−1)f[j] = f[j] + f[j-i] \quad (\text{mod } md-1)
    - 这里的 `md-1` 是因为它最终用在指数运算中。
        
4. 最终返回 `f`。
    

**作用**：计算**由小数（≤ sqrt(n)）组成的组合数**的方案数。

### 第二个 DP（计算 f1）

```cpp
auto f1=[](auto f1,auto g,auto b,auto _)->std::vector<int>{
    g[0]=f1[0]=1;
    return [](auto it,auto f1,auto g,auto b,auto _)->std::vector<int>{
        std::for_each(it.begin(),it.end(),[&](auto i){
            [&](auto it){
                std::for_each(it.begin(),it.end(),[&](auto j){
                    [&](auto &x,auto y){
                        return x=(x+y>=md-1?x+y-md+1:x+y),void();
                    }(g[j],g[j-i]);
                });
            }([](auto _,auto __)->std::vector<int>{
                return std::iota(_.begin(),_.end(),__),_;
            }(std::vector<int>(_-i+1),i));
            [&](auto it){
                std::for_each(it.begin(),it.end(),[&](auto j){
                    j+i*(b+1ll)<=_?
                    [&](auto &x,auto y){
                        return x=(x+y>=md-1?x+y-md+1:x+y),void();
                    }(f1[j+i*(b+1)],g[j])
                    :void();
                });
            }([](auto _,auto __)->std::vector<int>{
                return std::iota(_.begin(),_.end(),__),_;
            }(std::vector<int>(_+1),0));
        });
        return f1;
    }(
        [](auto _,auto __)->std::vector<int>{
            return std::iota(_.begin(),_.end(),__),_;
        }(std::vector<int>(b),1),
        f1,g,b,_
    );
}(
    std::vector<int>(mxn,0),std::vector<int>(mxn,0),
    (int)sqrt(*_),*_
);
```

这个比 `f` 复杂：

- `g` 是中间 DP，用来计算不超过某个上界的组合。
    
- 对每个 `i`（1 到 b-1）：
    
    1. 先更新 `g`（和 f 那边一样的完全背包）。
        
    2. 然后用 `g` 来更新 `f1`，但是这里物品是 `(b+1)` 以上的大数。
        
        - `j + i*(b+1) <= n` 时才更新。
            
- 最终 `f1` 存的就是**由大数（> sqrt(n)）组成的组合数**。

### 快速幂

```cpp
std::function<int(int,int)>func=[&](int x,int y){

};
```

- 经典递归快速幂：
    
    - `func(x,y) = x^y mod md`。

### 计算指数并输出

```cpp
std::cout<<func(
    *__,
    [&](auto it)->int{
        return std::accumulate(it.begin(),it.end(),0,[&](auto ans,auto i)->int{
            return (ans+(ll)f[i]*f1[*_-i])%(md-1);
        });

```

1. 构造 `it = [0, 1, 2, ..., n]`。
    
2. 对每个 `i`：
    
    - 累加 `f[i] * f1[n-i]`，对 `(md-1)` 取模。
        
3. 把这个累加和作为指数，底数是 `m` (`*__`)。
    
4. 用快速幂求 `m^(sum) mod md`。
    
5. 输出结果。

## **总结**

这个程序：

- 用 sqrt 分界把问题分成小数和大数两部分；
    
- 分别用 DP 求出方案数数组 `f` 和 `f1`；
    
- 然后通过卷积（`f[i]*f1[n-i]`）得到指数；
    
- 最终输出 `m^(指数) mod md`。
